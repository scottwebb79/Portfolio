Live and/or Simulated Event Data Entry

The purpose of this modal is to allow the user to input training events. There will be a form that allows the user to interact. Fields will include:

  • Syllabus
  • Trainee
  • Trainer
  • Environment
  • Conditions
  • Event Codes with Unsat - 5 Rating
  • Trainee Comments
  • Instructor Comments

The data entered here will be stored on the backend and tied directly to the trainee as well as the trainer.

Profile

Purpose: Users will have access to a profile page that allows them to review their personal information as well as achievements.

Dashboard

Purpose: Users will have a default, yet customizable, dashboard that will provide statistical data relating to the personnel within the users scope.

Message Center

Purpose: Each user will have an message center that will allow them to send and receive global E-Mails. In addition, this module will allow them to instant message amongst those who are logged on. If the receiving user is not online, the message will be waiting for them the next time they log on.

Calendar

Purpose: The calendar will allow user to schedule events, invite others to attend, and for administrators to schedule training and assign students to the training event. Notifications will be sent to users who are assigned to a training event.
