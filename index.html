Modal Live and/or Simulated

The live and/or simulated data entry module will allow training administrators to enter training and readiness event codes accomplished by a specifc trainee by way of a form. See below for requirement list

  • Trainee Name
  • Trainee Syllabus
  • Trainer Name
  • Trainer Instructor Designation
  • Date Accomplished
  • Input area for dropdown selection of accomplished event codes with radial selection for standard of Unsat to Proficient (4).
  • Exercise/Deployment Name
  • Student Comment input box
  • Instructor Comment input box
