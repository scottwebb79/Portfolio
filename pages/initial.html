MACCS Training Information System

Profile

Purpose: Users will have access to a profile page that allows them to review their personal information as well as achievements.

Dashboard

Purpose: Users will have a default, yet customizable, dashboard that will provide statistical data relating to the personnel within the users scope.

Message Center

Purpose: Each user will have an message center that will allow them to send and receive global E-Mails. In addition, this module will allow them to instant message amongst those who are logged on. If the receiving user is not online, the message will be waiting for them the next time they log on.
